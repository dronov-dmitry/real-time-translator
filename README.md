# Real-Time Translator

Приложение для перевода голоса в реальном времени. Всё работает на устройстве — без сервера и интернета.

## Возможности

- **Голосовой ввод** — говорите на одном языке, приложение переводит в текст на другом
- **Определение языка** — автоматически распознаёт, на каком языке вы говорите
- **Озвучивание перевода** — переведённый текст произносится вслух
- **Двусторонний режим** — поддержка обоих направлений перевода одновременно
- **Режим hands-free** — автоматическое распознавание пауз в речи (без кнопки)
- **Локальная работа** — STT, перевод и TTS выполняются полностью на устройстве
- **Уровни слов** — разбивка перевода по уровням сложности (A1–C1)

## Поддерживаемые языки

Английский, испанский, французский, немецкий — определение и перевод между всеми комбинациями.
